# 九宫格图片切割工具

一个简单易用的在线九宫格图片切割工具，帮助开发者快速将图片切分为九宫格，便于移动端开发实现自适应背景效果。

## ✨ 功能特色

### 核心功能
- 📁 **智能上传**：支持拖拽和点击上传，验证文件格式和大小
- ✂️ **精确切割**：使用HTML5 Canvas技术，将图片均分为9份
- 👁️ **实时预览**：显示原始图片和九宫格分割线
- 💾 **便捷下载**：支持单张下载和批量下载功能

### 界面设计
- 🎨 **专业美观**：现代化渐变色设计，卡片式布局
- 📱 **响应式设计**：完美适配桌面端、平板和手机端
- ⚡ **流畅体验**：平滑动画效果，实时进度提示
- 🔧 **用户友好**：清晰的操作流程和状态提示

## 🚀 使用方法

### 1. 上传图片
- 将图片拖拽到上传区域，或点击"选择图片"按钮
- 支持PNG、JPG格式，文件大小限制10MB

### 2. 预览切割
- 查看原始图片和九宫格分割线
- 点击"开始切割"按钮进行分割

### 3. 下载结果
- 单张下载：点击每张图片右上角的下载按钮
- 批量下载：点击"打包下载所有图片"按钮

## 🛠️ 技术栈

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **图片处理**：HTML5 Canvas API
- **文件处理**：File API + Blob
- **界面设计**：CSS Grid + Flexbox
- **交互体验**：ES6+ JavaScript

## 📝 开发说明

### 项目特点
- 纯前端实现，保护用户隐私
- 无需服务器，客户端直接处理图片
- 响应式设计，多设备兼容
- 现代化UI设计，用户体验优化

### 核心算法
使用Canvas API实现图片九宫格分割：
1. 计算每格子的精确尺寸
2. 遍历3×3网格进行切割
3. 保持原始图片质量和比例
4. 生成高质量的PNG格式输出
